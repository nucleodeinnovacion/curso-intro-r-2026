# Curso de introducción a R

## 🎯 Objetivo del curso

Brindar una base sólida para el uso de R en análisis de datos, cubriendo
desde la manipulación y exploración hasta la visualización y
comunicación de resultados.

## 📌 Requisitos

-    No se requieren conocimientos previos de programación

-    Se recomienda tener instalado R y RStudio antes del inicio
